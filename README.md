# Research Hub (LaTeX Template)

A modular LaTeX template to organize research work including literature, ideas, math notes, experiments, and logs in one place.

---

## Purpose

Research work often becomes scattered across different files and tools.  
This template helps keep everything organized in a single structured system.

---

## Structure

- `notes/` — general notes and concepts  
- `literature/` — paper summaries  
- `ideas/` — research ideas (one file per idea)  
- `math/` — derivations and proofs  
- `experiments/` — experiment plans and results  
- `logs/` — weekly progress tracking  
- `drafts/` — writing fragments  

---

## How to Use

**Ideas**  
Create a new file in `ideas/` for each idea.

**Literature**  
Create a new file in `literature/` for each paper.

**Experiments**  
Create a new file in `experiments/` for each experiment.

**Math**  
Use `math/` for derivations and proofs.

**Logs**  
Add a new file in `logs/` each week to track progress.

---

## Guidelines

- Keep entries short and clear  
- Use one file per item  
- Update regularly  
- Keep naming consistent  

---

## Goal

To maintain a clean, searchable, and organized research workflow.
